
# 📖 Online-Book-Borrowing-Platform

> **A Modern Cloud-Based Platform to Explore, Borrow, and Manage Books Online.** 

🚀 Technologies Used
Next.js
React.js
Tailwind CSS
Context API
JavaScript
MongoDB, 
Better-Auth,


## ✨ Features

### 🔐 Authentication (Better-Auth)
- **Secure Authentication:** Built-in secure login and signup system.
- **Session Control:** Robust client & server-side session tracking.
- **Protected Profiles:** Auto-redirects unauthenticated users to the login page.

### 🌐 Frontend (Next.js 14+ & React)
- **Dynamic Routing:** Seamless routing for specific book pages and user sub-routes (`/profile/update`).
- **Responsive Layout:** Beautiful UI designed for both desktop and mobile devices.
- **Global Image Support:** Configured remote patterns to accept images from any external URL.

### 💾 Database (MongoDB)
- **Real-time Book Fetching:** Dynamically loads the all-books section from the cloud.
- **Persistent Storage:** Safely stores user profile data and updating records.
